💎 Elghaly Tokenization Challenge

This project is an advanced practical application of Web3 skills and Smart Contract development on the Ethereum network, developed as part of the Speedrun Ethereum challenges.

🚀 Key Features

Advanced NFT System: Built a custom smart contract named "Elghaly" with the symbol "35" to ensure a unique project identity.
Public Minting: Added a feature allowing users to mint their own NFTs for a nominal Ethereum fee.
Secure Withdrawal System: A custom function allowing the contract owner to securely withdraw accumulated profits.
Public Network Compatibility: The project is configured for deployment on the Sepolia testnet to ensure a realistic user experience.
🛠️ Tech Stack

Solidity: For Smart Contract development.
Hardhat: For contract development and testing.
Next.js & Scaffold-ETH 2: For building a fast and responsive user interface.
OpenZeppelin: To ensure security according to ERC721 standards.
